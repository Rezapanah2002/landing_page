Exercises

  • Introduction: Write an HTML document that contains a brief introduction of yourself. Use different heading levels, paragraphs, and text formatting elements to make your introduction clear and attractive. Use at least one image to show your photo or something related to you. Save it as intro.html and open it in a web browser.
  • Book Review: Write an HTML document that contains a review of a movie, book, or game that you like or dislike. Use headings, paragraphs, lists, and images to make your review informative and engaging. Use the cite attribute to give credit to the source of the images or quotes. Save it as review.html and open it in a web browser.
  • Web Form: Create a HTML document that includes a form that collects the user's personal information
  • HTML Review: Create a web page that includes a navigation menu with four options. These options include Home, About Us, Services, and Contact.
  • LinkedIn html: In this exercise, we ask you to implement a website similar to LinkedIn.
  • Landing Page: The same page you are on :)
